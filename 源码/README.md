\# BlueSteelLight_miku

基于 Blue Steel Theme Pack 二次修改的 Visual Studio 亮色定制主题，内置专属界面配色、C++ 代码高亮配色与默认代码字体设置。



\## 主题信息

\- 原项目：Blue Steel Theme Pack

\- 开源协议：Apache 2.0

\- 修改定制：个人专属 C++ 美化配色

\- 适配：Visual Studio 2026



\## 包含内容

\- 自定义窗口界面底色、控件配色

\- 专属 C++ 语法高亮（运算符、new/delete、选中文本背景等）

\- 默认代码编辑器字体：JetBrains Mono 12号



\## 安装方法

1\. 下载项目 .vsix 文件

2\. 双击安装，按提示完成部署

3\. 重启 Visual Studio

4\. 工具 → 选项 → 环境 → 常规 → 颜色主题

&#x20;  选择：Custom Blue Steel Light\_miku



\## 字体说明

主题已内置字体配置为 JetBrains Mono 12号，

如需生效请自行安装该字体；未安装则自动 fallback 系统默认等宽字体，不影响正常使用。



\## 背景美化（可选）

本主题\*\*不含 ClaudiaIDE 任何文件\*\*，仅做配色与字体。

想要编辑器背景图效果：

1\. 自行安装 ClaudiaIDE 插件

2\. 在插件设置里手动指定背景图路径、透明度即可

3\. 本仓库不附带、不打包任何第三方插件文件

搭配背景教程（可选）
1. 安装 VS 插件：ClaudiaIDE
2. 下载本仓库里的 bg.png 壁纸（可能也叫“初音壁纸”按照实际为准）
3. 打开 VS → 工具 → 选项 → ClaudiaIDE
4. 按下面参数设置：
   - 图片路径：选择你下载的 bg.png
   - 透明度：0.35
   - 显示模式：Single 单张背景
   - 仅在编辑器内显示背景：开启
